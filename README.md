# Sarayu Kotha Portfolio

Static portfolio site for Sarayu Kotha.

## Publish

Publish this folder with any static-site host that serves `index.html` from the project root.

## Update Before Publishing

- Confirm whether the public email should remain visible.
- Add public project links once they are available.

## Local Preview

Open `index.html` directly in a browser, or run:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.
