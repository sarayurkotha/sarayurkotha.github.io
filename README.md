# Sarayu Kotha GitHub Pages Portfolio

Static GitHub Pages portfolio for Sarayu Kotha.

## Publish

1. Create or use a public GitHub repository under the `sarayurkotha` account.
2. Push this folder to the repository's `main` branch.
3. In GitHub, open **Settings > Pages** and confirm the source is **Deploy from a branch**, branch `main`, folder `/`.
4. If the repo is `sarayurkotha.github.io`, the site publishes at `https://sarayurkotha.github.io/`.
5. If the repo is `sarayukotha`, the site publishes at `https://sarayurkotha.github.io/sarayukotha/`.

## Update Before Publishing

- Confirm whether the public email should remain visible.
- Add any public GitHub repositories once they are available.
- Update the GitHub project cards after repositories are available.

## Local Preview

Open `index.html` directly in a browser, or run:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.
